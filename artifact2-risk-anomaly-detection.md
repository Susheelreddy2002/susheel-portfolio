# Risk & Anomaly Detection Using Regulatory Data

## Introduction

This project focused on identifying abnormal patterns and risk indicators within large regulatory datasets using statistical analysis and machine learning techniques.

## Description

The analysis involved reviewing more than 580,000 records to uncover unusual behaviors, detect anomalies, and identify trends associated with adverse events.

## Objective

To develop a scalable approach for detecting high-risk entities and supporting data-driven decision-making through advanced analytics.

## Process

1. Gathered and cleaned regulatory datasets.
2. Conducted exploratory data analysis.
3. Applied statistical methods to identify anomalies.
4. Built tree-based machine learning models.
5. Evaluated model performance.
6. Developed visual reports to communicate findings.

- Scikit-Learn
- Tableau
- Power BI
