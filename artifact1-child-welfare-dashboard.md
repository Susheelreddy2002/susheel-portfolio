# Child Welfare Risk Prediction Dashboard

## Introduction

This project focused on analyzing child welfare outcomes across U.S. states using predictive analytics and machine learning techniques. The goal was to identify key factors associated with child maltreatment risk and provide actionable insights for policymakers and nonprofit organizations.

## Description

The project was completed in partnership with Kids At Risk Action (KARA), a nonprofit organization dedicated to improving child welfare outcomes. State-level data from multiple public sources were integrated to develop predictive models and interactive visualizations that highlight areas of elevated risk.

## Objective

To develop a data-driven framework that identifies child welfare risk factors and supports evidence-based decision-making through predictive analytics and dashboard reporting.

## Process

1. Collected child welfare, demographic, and socioeconomic datasets from public sources.
2. Cleaned and standardized data for consistency and accuracy.
3. Performed exploratory data analysis to identify patterns and trends.
4. Engineered features relevant to child welfare outcomes.
5. Built and evaluated machine learning models.
6. Created interactive dashboards and visualizations.
7. Presented findings and policy recommendations.
